1. Установка БД:
1.1 helm repo add postgres https://andre128.github.io/OtusMagazin/
1.2 helm install postgres postgres/postgres --create-namespace --namespace m

2. Установка приложений:
2.1.1 helm repo add app https://andre128.github.io/OtusMagazin/
2.1.2 helm install app app/app
2.2.1 helm repo add auth/auth https://andre128.github.io/OtusMagazin/
2.2.2 helm install auth auth/auth
2.3 helm repo update

3. Убедиться, что ingress-controller установлен
4. Убедиться, что в /etc/hosts/ прописан ip ingress controller 
ip_ingress_controller arch.homework

5. Проверка с помощью newman и postman-collection
5.1 Скачать коллекцию: 
https://github.com/Andre128/OtusMagazin/blob/48058b56d9aca44fa2a454bd29a8a520a3af6520/Auth_service.postman_collection.json

5.2 Запустить newman run Auth_service.postman_collection.json --verbose
